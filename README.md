## Install and Usage

to install this script

- Copy this repostiroty
- Open the terminal.
- Type te following commands:
```
chmod +x ./dotmk.sh
./dotmk.sh
```
- If u want install run 
```
./dotmk.sh -i 
```
- To see help run 
```
./dotmk.sh -h 
```
